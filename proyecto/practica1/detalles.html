Nuestra aplicación web consistirá en una red social de publicación de ofertas de tiendas de primera mano principalemente.Además, los usuarios podrán publicar productos propios para que los administradores puedan comprarlo, para posteriormente venderlo en la web. Solo los administradores podrán publicar productos de segunda mano comprados a otros usuarios. Las ofertas podrán ser tanto objetos físicos como ropa, electrónica o mobiliario, y también ofertas electrónicas como suscripciones a sitios, cupones descuentos o videojuegos. La aplicación te permitirá tener en un control de las ofertas de distintas tiendas y/o marcas. Dentro de cada oferta se podrán poner comentarios para compartir o comentar la experiencia de las ofertas y productos.

TIPOS DE USUARIOS
Usuario Funcionalidad de usuario
Administrador Tendrá el control total sobre la página web, cambiar su aspecto y gestionar los demás usuarios.
Empresa verificada Usuario que se tiene la certeza que se corresponde con la empresa que dice ser y que puede ponersus propias ofertas y comentar en otras
Moderador Usuario encargado de comprobar que las ofertas publicadas por otros usuarios son reales y que no incumplen la normativa de la página.
Usuario Premium Usuario que tiene acceso anticipado a ciertas ofertas.
Usuario registrado Usuario con posibilidad de publicar sus propias ofertas y de comentar.
Usuario no registrado Usuario que únicamente puede visualizar las ofertas, pero no puede registrar ninguna ni publicar prodcutos para venderlos en segunda mano.
